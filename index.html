Frogger Ninja : 3D Multiplayer Online Game

FroggerNinja is a prototype of classic game of 80s “Frogger”, which was a single player game with objective to make the frog be able to cross the street full of heavy traffic. The project extends the game into a multiplayer version using SocketIO. The game is developed in ThreeJS framework. The technology underneath it is WebGL, an extension to OpenGL 2.0. Charting libraries such as Google HighCharts and D3 JS are used. The project makes use of a physics engine named PhysiJS.

Profils-SJSU

Profils

‘Profils’ is an android application, which uses the concept of Geo Fencing. It automatically enables or disables the ringer modes of the phone if it enters specific location. For example, whenever you go to school, your phone will be automatically put on silent mode and once you come out it will set back to the previous ringing mode. Goal of the app is to make user free of setting the profile each time. This app uses Network Provider or Wi-Fi to get your current location and changes the way your phone behaves. So, you don’t have to remember things.

Technologies used => Android, Geo-fencing, NoSQL in android, GSON, GoogleMapsV3.0
